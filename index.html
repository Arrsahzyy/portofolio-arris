Tentang Saya

Mahasiswa Teknik Elektro, Institut Teknologi Sumatera

Saat ini saya menempuh semester 5 di jurusan Teknik Elektro dengan fokus pada Sistem Tenaga, Arus Kuat dan Sistem Kendali. Saya sangat tertarik pada persimpangan antara perangkat keras dan perangkat lunak, dan selalu antusias untuk belajar hal baru.

Di luar akademis, saya aktif mengembangkan kemampuan praktis melalui berbagai kegiatan: sebagai anggota tim robotika (UROTERA) dan tim pengembangan UAV (Swarnakasa), saya mengasah keterampilan kerja tim, manajemen proyek, dan pemecahan masalah. Selain itu, keterlibatan saya di Dewan Energi Lampung memperkuat kemampuan saya dalam riset dan kolaborasi tim untuk mengkaji potensi energi di wilayah Lampung.
